pyart.correct.phase_proc.LP_solver¶

pyart.correct.phase_proc.LP_solver(A_Matrix, B_vectors, weights, it_lim=7000, presolve=True, really_verbose=False)[source]¶

Solve the Linear Programming problem, see Giangrande et al, 2012.

Parameters :

A_Matrix : matrix

Row augmented A matrix, see construct_A_matrix

B_vectors : matrix

Matrix containing B vectors, see construct_B_vectors

weights : array

Weights.

it_lim : int

Simplex iteration limit.

presolve : bool

True to use the LP presolver.

really_verbose : bool

True to print LPX messaging. False to suppress.
Returns :

soln : array

Solution to LP problem.
