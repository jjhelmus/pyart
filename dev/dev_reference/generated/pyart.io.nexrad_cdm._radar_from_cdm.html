pyart.io.nexrad_cdm._radar_from_cdm¶

pyart.io.nexrad_cdm._radar_from_cdm(dataset, vnames, field_mapping, field_metadata)[source]¶

Return a Radar from a CDM dataset.

Parameters :

dataset : Dataset

CDM dataset with NEXRAD Level II data.

vnames : dict

Variable and dimension names in dataset which will be used to populate the Radar.

field_mapping : dict

Mapping between CDM variable and radar field names. See read_nexrad_cdm for details.

field_metadata :

Metadata for the field specified in field_mapping. See read_nexrad_cdm for details.
Returns :

radar : Radar

Radar containing the requested fields.
